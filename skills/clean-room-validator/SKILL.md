# Clean Room Validator
## 核心能力
验证 AI 生成的部署教程是否真正可行，确保零基础用户能按教程无障碍完成部署。

## 执行流程
### 1. 解析教程
读取 Markdown 教程文件，提取所有代码块中的命令。

### 2. 创建干净环境
在 AutoDL 服务器上创建一个全新的 Python 虚拟环境和独立工作目录，模拟小白用户的初始状态。

### 3. 逐步执行命令
在干净环境中按顺序执行每条命令，记录执行结果。核心原则：只执行教程中明确写出的命令，不做任何额外推理或补救。

### 4. 生成验证报告
生成包含每一步执行结果的 Markdown 报告。

### 5. 修补教程(如有失败步骤)
调用 DeepSeek 分析失败原因，生成修补建议。

### 6. 清理
删除虚拟环境和临时目录。
