## 一、项目概述
### 1.1 项目背景与动机
在服务器运维场景中，新手部署流程经常出现不规范操作、安全漏洞、环境污染问题；人工审核部署教程效率低，无法批量自动化检测风险。
传统部署方案依赖人工逐行核对命令，容易遗漏权限、防火墙、日志审计、依赖隔离等安全要点；零基础使用者照搬网上教程极易把高危配置带入生产环境。
本项目搭建一套多 Agent 自动化运维助手体系，实现部署生成 - 自动执行 - 安全审计校验全闭环，替代人工重复操作，降低部署安全风险与上手门槛。
### 1.2 项目目标
搭建三套联动 Skill 组件，形成完整部署工作流：进度推送、自动部署、安全验证；
实现一键生成 Ubuntu 服务部署脚本、自动执行环境安装；
接入 DeepSeek 大模型，对部署文档做全自动安全漏洞审计，输出标准化修复方案；
适配 OpenClaw 多 Agent 调度框架，支持后台网关常驻运行；
面向零基础用户，屏蔽复杂命令与安全配置细节，输出可直接落地的安全部署 SOP。
### 1.3 创新点
- 不是信息搬运，而是自动实操
- 教程闭环验证机制
- 多 Agent 协作
- 面向零基础用户
## 二、系统设计
### 2.1 整体架构
调度层：OpenClaw Gateway 作为核心调度中心，管理三个 Skill 生命周期、进程启停、环境变量分发；
业务 Skill 层：
Skill1：每日项目雷达（消息推送、进度通知、任务状态上报）
Skill2：自动部署引擎（读取配置、执行 apt/pip 环境安装、拉起服务）
Skill3：干净房间验证（读取 md 部署文档 → 请求 DeepSeek API → 解析漏洞 → 输出 Markdown 安全报告）
底层依赖层：Ubuntu 24.04 系统、Python3、requests 网络库、DeepSeek 大模型 API、systemd 进程托管。
### 2.2 技术栈
系统：阿里云 ECS Ubuntu 24.04 LTS
运行语言：Python3
AI 能力：DeepSeek v4-flash API（安全审计推理）
调度框架：OpenClaw（多 Skill 网关、配置隔离、profile 多实例）
工具库：requests（HTTP 请求）、traceback（异常捕获）、systemd（服务常驻）
文档格式：Markdown（部署文档、安全报告、作业文档）

#### Skill 1：每日项目雷达
功能描述
定时 / 手动推送项目开发进度、部署状态、校验结果；向用户反馈脚本运行成功 / 失败、漏洞数量、修复要点；支持任务节点消息提醒。
设计思路

设计思路
分阶段执行：系统初始化→运行时依赖→应用启动；增加执行超时、错误捕获，单步骤失败不终止全流程，记录错误栈便于排查。

## 三、实现过程
### 3.1 环境搭建
初始环境问题 1
问题：Windows PowerShell 识别不了export Linux 环境变量指令
方案：切换 SSH 远程登录阿里云 ECS，全程使用 root Shell 操作，Linux 原生语法配置 API 密钥。
初始环境问题 2
问题：find 检索显示教程文件存在，但 cat 读取提示路径不存在
方案：严格匹配文件夹拼写tutorialials，手动mkdir -p创建完整层级目录，再写入教程文本。
初始环境问题 3
问题：Vim 编辑器卡死、输入指令无响应、终端滴滴告警
方案：放弃 Vim 编辑，使用cat > 文件名 <<'EOF'一键整段覆盖写入代码，彻底规避编辑器阻塞；后台kill %强制终止挂起 Vim 进程。
初始环境问题 4
问题：原版校验脚本异常捕获屏蔽报错，运行后空白无输出
方案：重写全新完整脚本，每一层读写、API 请求、入口都添加 traceback 异常打印，实现错误可见。
初始环境问题 5
问题：复制 Skill 到 OpenClaw 工作区提示目录不存在
方案：提前执行mkdir -p递归创建完整.openclaw-bing/workspace/skills/路径，再拷贝文件夹；网关执行systemd服务重启加载技能。

## 五、云服务器部署说明
### 5.1 服务器选型与对比
选用阿里云 ECS 共享型实例，对比三大主流平台：
阿里云：国内访问延迟低、新手引导完善、学生优惠力度大；适合课程作业、轻量 AI 接口调度；
腾讯云：带宽定价略高，轻量实例磁盘配额更小；
华为云：企业级服务完善，但个人学生套餐门槛更高。
本项目仅运行 Python 轻量脚本 + API 转发，阿里云最低配实例性能完全富余。
### 5.2 部署方案
采用单机一体化部署：同一台 ECS 同时承载 OpenClaw 网关、三个 Skill 运行环境、md 文档存储；无多机拆分，降低运维复杂度，适配课程作业体量；systemd 托管openclaw-gateway-bing服务，开机自启后台常驻。
### 5.3 成本分析
阿里云学生机套餐：按月计费，单实例月度费用约9.9 元；无额外带宽、流量超额支出；DeepSeek API 免费额度足够支撑作业测试，零推理额外开销；整体月实际总成本控制在 10 元以内。
